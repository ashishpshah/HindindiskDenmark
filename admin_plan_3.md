# Admin Panel Scope

The admin panel will allow the restaurant owner and their team to manage all aspects of the website without requiring technical knowledge. It will be accessible via a secure login from any device.

## Admin Module Features & Capabilities

- **Dashboard**: Summary of today's orders, new reservations, total revenue, pending items, and quick action links.
- **Branch Management**: Add/edit branch details, address, phone, opening hours, delivery settings, and status.
- **Menu Management**: Add/edit/delete dishes, set categories, upload photos, set prices, allergen labels, dietary tags, branch availability, and active/inactive status.
- **Order Management**: View all orders, update order status (Accepted/Preparing/Ready/Completed/Cancelled), filter by branch/date/status.
- **Reservation Management**: View table bookings, confirm or cancel reservations, filter by branch/date, add internal notes.
- **Customer Management**: View registered customers, order history per customer, manage accounts if needed.
- **Offers & Coupons**: Create and manage promotional offers and coupon codes with validity and rules.
- **Gallery Management**: Upload, organize, and remove photos for the public Gallery page.
- **Review Management**: Manage featured reviews or testimonials displayed on the website.
- **Content Management**: Edit Home page text, About page content, Contact details, and footer information.
- **Language Content**: Manage Danish and English translations for menu items, pages, and website content.
- **Settings**: Manage payment gateway settings, GDPR/cookie settings, notification emails, and website preferences.
- **Reports**: Basic reports: total orders, revenue by branch, most ordered dishes, reservation count, and customer growth.
